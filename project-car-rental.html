Car Rental Web Application

Personal Project – React, TypeScript, Tailwind CSS

Mount Motors Car Rental is a fully responsive web application built to showcase a premium, modern car booking experience. The project emphasizes performance, clean visuals, and usability. I designed and developed the site from scratch using React and TypeScript with the Next.js App Router, structured around reusable components.

The website includes sections like Hero, Top Cars, Categories, Testimonials, and a dynamic Reviews page with user feedback. A Reservation system was built to manage booked cars, with a custom Reservation Summary Modal and Checkout flow. Users can edit or delete their bookings directly through a clean UI.

The goal was to simulate a real-world booking experience, while focusing on front-end performance and interface polish using Tailwind CSS, clean grid layouts, icons, hover animations, and responsive cards. It also includes a dashboard layout and routing logic, and the project continues to expand with features like an admin panel and availability filters.
