Smush Burger Restaurant Website

Freelance Project – HTML, CSS, JavaScript

Smush Burger is a vibrant and modern restaurant specializing in smash burgers, known for bold flavors and edgy branding. I was hired as a freelance developer to create a dynamic and visually appealing website to reflect their identity.

The site includes smooth animations, a responsive layout, and interactive elements to enhance user experience. I worked closely with the brand’s visual direction to ensure the design aligned with their unique aesthetic. Every section was crafted to be simple, fun, and intuitive for customers.

Built entirely with HTML, CSS, and JavaScript, the website showcases my ability to design and code fast, clean, responsive restaurant interfaces from scratch.
